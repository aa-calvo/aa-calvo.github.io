# Akros Drone

# Goals

The primary objective of this project is learning! This means is that I may "reinvent the wheel" sometimes. The decision to build my own "plant model" in C++ is the main example of this.

The secondary objectives are driven by personal curiosity and career development. These will drive some of the design decisions, but should not lead to large deviations from the primary objections.

Primary Learning Objectives:

  • Analytical Dynamics
    • Equations of Motion
    • Coordinate Transformations
  • Control Theory
    • Basic PID control
    • Gain Margins
  • Robot Geometry
    • Quaternions

Secondary Learning Objectives:

  • Unreal Engine
  • Real-Time Linux (PREEMPT_RT)
  • Machine Learning
    • Object detection
    • Reinforcement learning

# Autonomy

Machine learning is a secondary objective, therefore I will keep the first autonomy milestones as simple as possible. This means is focusing more on the "hard maths" instead of trying to use ML/RL to magically complete our goal.

# Milestones

  1. First Flight (Gazeebo)
    • Use Gazeebo simulation environment to test basic take-off and landing
  2. First Flight (Real)
    • Make sure our chosen hardware and software stack actually work
  3. First Flight (Our Sim)
    • Develop our custom flight simulator
